# 🎯 AML 洗錢防制排位賽系統

2026 年度洗錢防制教育訓練 — 線上即時排位賽系統。

---

## 📁 檔案結構

```
aml-quiz-game/
├── server.js          ← 主程式（Node.js 伺服器）
├── questions.json     ← 題目檔案（可自行新增/修改）
├── package.json
└── public/
    ├── index.html     ← 員工參賽頁面（手機開啟）
    ├── admin.html     ← 主持人控制台（投影機電腦開啟）
    └── style.css      ← 樣式
```

---

## 🚀 第一次使用：安裝步驟

> 只需做一次。

1. 打開 **PowerShell 或終端機**，進入此資料夾：
   ```
   cd "C:\路徑\aml-quiz-game"
   ```

2. 安裝必要套件：
   ```
   npm install
   ```

---

## ▶ 每次使用：啟動步驟

1. 啟動伺服器：
   ```
   node server.js
   ```

2. 終端機會顯示類似下方的訊息：
   ```
   📱 員工加入網址：http://192.168.1.x:3000
   🎮 主持人控制台：http://192.168.1.x:3000/admin.html
   ```

3. **主持人電腦**（投影給大家看）：開啟 `admin.html` 網址
4. **員工**（各自手機）：掃 QR Code 或輸入 `index.html` 網址

> ⚠️ 所有設備需連接同一個 WiFi，否則無法連線。

## 🎮 遊戲流程

```
主持人                          員工（手機）
  │                                │
  ├─ 開啟 admin.html               ├─ 掃 QR Code 開啟遊戲頁
  │                                ├─ 輸入姓名加入
  │  看到玩家加入                   │
  │                                │  等待大廳畫面
  ├─ 按「開始遊戲」                 │
  │                                ├─ 看到題目 + 倒數計時
  │  看到作答進度 X/Y              ├─ 點選答案
  │                                │
  ├─ 全部作答後按「揭曉答案」        │  看到「答對/答錯 + 得分」
  │                                │
  │  投影片顯示解析 + 排行榜        ├─ 看到解析 + 目前排行
  │                                │
  ├─ 按「下一題」（重複以上）       │
  │                                │
  ├─ 最後一題後按「結束遊戲」       ├─ 看到最終排行榜
  │                                │
  └─ 按「下載 CSV」匯出成績        └─
```

## 📝 如何修改題目

打開 `questions.json`，每題格式如下：

```json
{
  "id": 1,
  "category": "📗 基本常識",
  "text": "題目內容（可用 \n 換行）",
  "options": [
    "A）選項一",
    "B）選項二",
    "C）選項三",
    "D）選項四"
  ],
  "correct": "B",
  "explanation": "解析說明文字...",
  "timeLimit": 60,
  "isMultiple": false
}
```

| 欄位 | 說明 |
|------|------|
| `category` | 題目分類（顯示在題目上方）|
| `text` | 題目內容 |
| `options` | 四個選項（固定 A~D）|
| `correct` | 正確答案（單選填 `"B"`，複選填 `["A","C"]`）|
| `explanation` | 揭曉後顯示的解析說明 |
| `timeLimit` | 作答時間（秒）|
| `isMultiple` | `true` = 複選題，`false` = 單選題 |

## 🏆 計分規則

| 狀況 | 得分 |
|------|------|
| 答對（第 1 秒內）| 約 1,000 分 |
| 答對（時間用完前）| 約 500~1,000 分（依速度） |
| 答錯 | 0 分 |

越快答對，得分越高！

---

## 📊 匯出成績

遊戲結束後，主持人控制台可按「下載 CSV」按鈕，匯出包含 **名次、姓名、得分** 的完訓紀錄，可直接用 Excel 開啟。

## 🔧 常見問題

**Q：員工說無法開啟網頁？**
A：確認大家連的 WiFi 和主持人電腦是同一個。終端機顯示的 IP 就是員工要輸入的網址。

**Q：伺服器關掉怎麼辦？**
A：重新執行 `node server.js` 即可，但遊戲分數會重置。

**Q：可以在家測試嗎？**
A：可以。同一台電腦開啟 `http://localhost:3000`（員工頁）和 `http://localhost:3000/admin.html`（主持人頁）即可模擬。

**Q：要怎麼停止伺服器？**
A：在終端機按 `Ctrl + C`。
